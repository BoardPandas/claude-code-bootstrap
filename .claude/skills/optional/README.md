# Optional Skills

These skills are for specific technologies or domains. Include the ones relevant to your project.

## How to Enable

1. Keep the skill files you need in this directory
2. Add corresponding entries to `../../skill-rules.json` (copy from `../../skill-rules.examples.json`)
3. Delete skill files you don't need

## Available Optional Skills

| Skill | For Projects Using |
|-------|-------------------|
| `UIC_Guidelines.md` | React/JSX with UI element tracking |
| `go-desktop-agent-guidelines.md` | Go + Wails desktop apps |
| `openai-api-expert.md` | OpenAI GPT API integration |
| `prompt-engineering-expert.md` | AI prompt design |
| `rest-api-expert.md` | REST API design patterns |
